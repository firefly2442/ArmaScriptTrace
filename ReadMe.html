Introduction

ArmaScriptTrace provides the ability to parse Arma2 script files and display an image of the calls to other scripts. It will follow execfsm, preprocessfilelinenumbers, addaction, preprocessfile, execvm, and #include calls.

Supported Filetypes:
  • .sqf
  • .sqs
  • .ext
  • .sqm
  • .hpp
  • .fsm
  • .h
  • .cpp

System Requirements
  • Java Runtime Environment

How To Use ArmaScriptTrace
  • Double click the .jar file (or the batch file) to open the application.
  • Click the file menu and open a script file, or alternatively, drag and drop scripts onto the main window.
  • Optionally, save the resulting image on your computer.
